# Contributing

## Getting Started

1. Fork the repository
2. Clone your fork
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b your-feature`

## Development

```bash
npm start
```

The server runs on `http://127.0.0.1:3000` by default.

## Cloudflare Worker

```bash
npx wrangler dev
```

## Pull Requests

- Keep changes focused and minimal
- Test your changes before submitting
- Write clear commit messages
- Update documentation if adding/changing endpoints

## Code Style

- Use tabs for indentation
- Follow existing patterns in the codebase
- Keep routes clean and consistent with the Express.js version

## Reporting Issues

Open an issue with:
- Steps to reproduce
- Expected behavior
- Actual behavior
